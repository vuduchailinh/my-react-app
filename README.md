# My React App

### How to run the app
#### 1. Start the server

```bash
json-server --watch db.json --port 3001
```

#### 2. Start the app

```bash
npm start
```

### 2. What is the app about?
This app is a simple React app that displays a list of users and their details. 
The app is built using:
- React 
- React Router 
- React Query 
- Axios
- Radix UI
- Storybook
